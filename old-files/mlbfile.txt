MLB (Masdjab Library) File

Header
Offs  Size      Usage
0000  3 bytes   Signature, "MJL" for library, "MJX" for executable
0003  1 byte    Reserved
0004  DWord     File size
0008  Word      Header size
000A  Word      Reserved
000C  Word      Section table offset
000E  Word      Section table size
0010  DWord     Main entry point
0014  DWord     Checksum


Section Table
0000  4 Byte    Section name
0004  Byte      Section characteristic
                1 = code
                2 = text (string table)
                3 = data/resource
                4 = relocation table
                5 = import table
                6 = export table
                7 = app info
0005  3 bytes   Reserved
0008  DWord     Section offset
000C  DWord     Section size


Import Table
'kernel.dll', 0, 'GetUserInfo', 0, 'GetProcAddress', 0, 0

Export Table
'puts', 0, 0000, 'waitkey', 0, 004b, 'exitprocess', 0, 002e, 0

String Table Section
length, text, length, text, 0

Relocation Table Section
offset, offset, offset, ...
